# PeacefulParents

PeacefulParents is a calm, local-first family care command center for tired parents. It includes baby care logs, shared tasks, whose-turn rotation, supplies, settings, and a peaceful audio space.

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Notes

- No backend, auth, payment, Plaid, or external APIs are required.
- Data is stored locally in the browser for the MVP.
- Audio tracks in `public/audio/` are placeholders and should be replaced with royalty-free or public-domain files before production use.
